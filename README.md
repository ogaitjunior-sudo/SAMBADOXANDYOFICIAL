# Samba do Xandy

Landing page em React + Vite para apresentar o projeto, destacar o canal oficial e receber pedidos de orcamento por e-mail.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run lint`

## Conteudo do site

Os textos, links principais e informacoes de contato ficam centralizados em [src/content/siteContent.ts](/C:/Users/bangu/OneDrive/Área%20de%20Trabalho/samba-vibes-site-main/src/content/siteContent.ts:1).

## Fluxo de contato

O formulario gera um rascunho de e-mail com os dados preenchidos pelo visitante. Assim o site continua funcional sem backend e sem depender de numero de WhatsApp placeholder.
